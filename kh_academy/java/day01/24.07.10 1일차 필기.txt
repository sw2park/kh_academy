1. 프로그래밍 언어
 1-1. 프로그래밍 언어란?
  - 프로그래밍 언어란 주어진 어떤 문제를 해결하기 위해 인간과 컴퓨터 사이에서 의사소통을 가능케 하는 인공적인 언어
  - 이 언어를 통해서 사용자는 컴퓨터에게 일련의 일을 시키는 명령어들의 집합체인 프로그램을 작성할 수 있다.

2. java 언어 소개 (java 언어 특징)
 - 1996년 1월 릴리즈 
 - 6개월마다 신규 버전 업데이트
 2-1. 운영체제(OS)에서의 독립적 성향을 갖고있음
  - JVM이 설치된 환경이라면 어디서든지 실행 가능하다. 
   ㄴ C언어와 달리 OS 호환성이 높음
 2-2. 객체지향 언어 (강사쌤 추천 책 : 객체지향의 사실과 오해)
  - 상속, 캡슐화, 다형성
  - 코드의 재사용 및 유지보수에 용이함(프레임워크 공부 시 체감될 예정)
 2-3. 자동 메모리 관리
  - Gabage Collector가 자동으로 메모리를 관리해준다. (C언어는 개발자가 직접 처리해야했으나 그 과정을 개선하여 나온것이 java)
 2-4. 네트워크, 분산처리, 멀티스레드
  - 시스템과 관계없이 네트워크, 분산처리, 멀티스레드 구현을 위한 손쉬운 API 제공

3. java 가상머신(Java Virtual Machine / JVM)
 - 컴파일된 자바 바이트코드를 실행시켜 주는 소프트웨어
 - 자바 프로그램은 JVM이 설치된 환경이라면 운영체제나 하드웨어에 종속되지 않고 실행이 가능하다.
 - 운영체제로부터 독립적이다.

4. 컴파일(Compile)
 - Hello.java(소스코드) ---------> 컴파일(Compile) / 컴파일러(Compiler) -------> Hello.class
 - 컴파일러(Compiler) : 소스코드를 프로그램으로 변환해주는 프로그램
 4-1. 컴파일 과정의 단점
   - 컴파일의 결과물로 만들어진 프로그램이 실행되어지는 환경(운영체제 등)에 따라 개별적인 컴파일이 이루어져야 한다.

5. 자바설치
 5-1. 내 pc에 자바가 설치가 되어있는지 여부 확인
 - 명령어로 확인
 - 실행창 > cmd > java -version (java 설치되어있는지 확인)
 - 실행창 > cmd > javac (java 컴파일러 설치되어있는지 확인)
 5-2. 자바설치
